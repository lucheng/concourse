Google 周四在 Twitter 上宣布，Chromebook 笔记本已经开始接受预订，大家可以通过 Amazon 和 Best Buy 进行预订。这批由三星和宏基代工的笔记本预计将在6月15日发出。 这次发布的笔记本主要是希望了解美国消费者是否对这样的一个新操作系统感兴趣，以及是否会常用 Web 应用程序。 Amazon 上的促销信息是这样描述的： “Chromebooks 对 用户最为常用的 Web 功能进行了优化。相比普通的笔记本，你可以在 Chromebook 上体验到更快，更便捷，更安全的 Web 功能” Chromebook 笔记本的价格大致上与 Windows-7 系统的笔记本电脑持平，大约是$379.99 至 $499.99. Google 在上网本以及笔记本电脑方面的尝试不如它在 Android 设备方面的尝试那么流畅。Google 公司最早宣布将要开发基于 Linux 的笔记本电脑可以追溯到2009年7月，但时至今日我们才看到这款 Chrome 系统的笔记本电脑，是不是意味着公司对于 Chrome OS项目的推进并不像推进 Android 的手机和平板电脑开发那么给力呢？ 比较有趣的是，Google正向商务人士和学校提供一项订阅服务。商务人士可以通过每月支付 $28 美金来使用这项服务，而学生仅需支付 $20 美金。这个价格包含了支持，更新，安全等等服务。 个人认为 Google 将笔记本电脑的体验放在 Web 应用的基础上并非一个明智的选择，因为从用户的角度，尽管 Web 功能非常常用，大家更希望在笔记本电脑上去使用一些非 Web 的应用程序。比如你想要在这款笔记本电脑上使用相片编辑，你就必须得在线使用才能OK。不知道是不是所有的用户都能够接受这样一种模式了。 你打算预订一台来自 Google 的 Chromebook 吗？
