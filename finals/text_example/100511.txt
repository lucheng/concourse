微软的大作Windows 8在发布的时候到底会有几个版本？这一直是人们热议的话题。 今天，谜底终于解开了。此前Windows的版本让人眼花缭乱，而Windows 8简化了整个产品线，将推出三个SKU版本：Windows 8、Windows 8 Pro和Windows RT。（注：如果把特殊的企业版和中国特别版算上的话，SKU应该是五个，但常规产品线只有这三个版本）。 Windows 8是面向用户的基础版产品，拥有Windows 8的基础功能。 Windows 8 Pro则添加了加密、虚拟化、PC管理工具和域连接能力，还带有Windows Media Center。 Windows RT则是ARM平台下的独立版本，预计平板设备将会安装这款系统，可运行针对触控优化过的Word, Excel和PowerPoint等应用。 尽管这样的“阵容”仍没有苹果的那么纯粹，但与Windows 7相比，已算得上是大删减了。 via TC
