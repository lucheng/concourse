我是草 
 &nbsp; 
 &nbsp; 
 远了，远了，只剩下苍老了 
 我泥土的心上已长出了草 
 我额上的汗在一滴滴乡愁的燃烧 
 我想跟着羊回到属于我的青草根下 
 让一根草在苍茫的大地把我埋葬 
 &nbsp; 
 我纯洁，纯洁得像一种病一样 
 只要一切，一切能把我粉碎，我就随风而飞 
 只要我，我粉碎了一切，我就和月亮举杯 
 &nbsp; 
 我是一个草地上长大的孩子 
 温暖我的是那些已经永恒的文字 
 我死了，我就做一根草，吐出如诗的珠子 
 &nbsp; 
 诗是孤独的，我也是孤独的 
 我不传统，也不技巧，只是泥土中长出的一根草 
 为了对得起我将被腐烂我将被遗忘我将被死亡 
 我要像盐一样，在人间这盘菜里留下自己的味道 
 我要像草一样，每天在长高，每天在苍老 
 我要守着自己，守着自己的一点地，哪怕如针尖那般小 
 我要如太阳一样，吼出自己灿烂的光芒 
 &nbsp; 
 我要用泪水洗亮我的世界，用汗水洗亮我的名字 
 我不接受选择，哪怕我如草一样连根拔去 
 &nbsp; 
 我是永远的远方，我的世界是如此荒凉 
 我这辈子只干好一件事，只好好做一根草 
 &nbsp; 
 失去的将失去，失去后留下的是一个黑黑的底 
 我假如有一天成了一棵白菜 
 我也会从白菜里让你吃出鸡与鱼 
 &nbsp; 
 我是一根草，来自大地的深处 
 我两手空空，只要有白菜与大米，我已足够 
 我泥土的心里只有泥土 
 &nbsp; 
 那么多人，把身体如手一样弯向大地 
 那么多泪水，流成了大地上的河流 
 只有草一辈子不离开自己小小的土地 
 &nbsp; 
 一棵草有自己的祖国 
 一根草在大地上开花结果 
 一根草在大地上生长着我 
 在一根草下躺下吧，草也是大地 
 &nbsp; 
 一个农民抱着他的粮食失声痛苦 
 我痛哭一根草的生命却还在冬天里活着 
 它在土地里满怀着乡愁，它在土地里等待着燃烧 
 它在等待春天，把自己的草命再次燃烧成一根草 
 &nbsp; 
 我是一滴祖国的血，也是祖国的一根草 
 我的生命是来自泥土流淌着的乳汁 
 我知道，必须有一根草，才能有另一朵花 
 &nbsp; 
 我始终在路上，在飘的路上去接近自己的零乡 
 我始终在失败，在为败而斗中让大地胜利 
 我本身就是民间，我本身就是一棵有虫子的草 
 我如草一样，只在晨光中闪烁自己的梦露之国 
 不管是苦是甜，只为活着的每一分每一秒 
 只为活回真实的自己，尽管大部分时间自己已死去 
 &nbsp; 
 上辈子的我死于一场电影 
 这一辈子的我为一场电影而生 
 我就是草的电影，我就是电影 
 &nbsp; 
 失去了泥土的草，它还是不是草 
 我是一个新人，每个黑夜如母亲一样让我重新出生 
 我是一个错字，一个没谁认识的错字 
 在滔滔众口的场面一闪就该消失 
 &nbsp; 
 天是我的穹顶，山是我的座椅 
 因为爱，我被定义为人 
 当我成为大地上一捧泥土上的草 
 我要感谢天下的农民与工人，他们只当我是一个普通的陌生人 
 我要感谢我的祖国，用阳光、风、雨，使我成长为一棵草 
 我要感谢那些睡在街边的人，睡在车站的人，睡在树底下的人，他们像草一样，睡在大地上任何一个地方 
 我要感谢那些在没有路的地方走出路的人 
 让我踏出这些路，去任何一个我想去的地方 
 &nbsp; 
 我高兴，我死了之后，能够死在路上 
 我高兴，我死了之后能够成为一根草 
 我要好好劳动，对得起我劳作一生的祖国 
 &nbsp; 
 一根草，冬天只能在根中保留自己的生命 
 一本书，是一根草的心灵城市 
 一本书里，有一个人向往的幸福王国 
 书里，有无数梦想的虫子，有无数梦想的文字 
 &nbsp; 
 很多的门，我只能有一扇 
 很多的路，我只能走一条 
 抵达远方的路，不在我的脚下，在一根草下 
 &nbsp; 
 把自己当作草，就有秋的火来烧 
 我穷，只有一支笔，一张纸 
 我在穷尽一切心机做着一件吃不饱的事 
 我是一棵草，一根用良心滋养的草 
 只有草能生长在帝王对面，拿出大地全部的声音，用尽一生的血与泪水，向天地证明它是一个顶天立地的诗人 
 &nbsp; 
 我是草，我的爱很小很小 
 我丢掉了语言，仿佛一辆空荡荡的马车 
 从一个秋天奔向另一个死亡的秋天 
 &nbsp; 
 风，已剥尽了我，我已虚无 
 我已没有家，没有亲人 
 我只在云朵和雪山深处，在泥土上 
 在心不值钱的春天，我用了整整一生才成为一棵草 
 &nbsp; 
 我对你的爱，至少还有三千里路没有完成 
 我只能用赤裸干净的灵魂照亮你的眼睛 
 我只能用文字做一盏灯 
 挂在一根草上，挂在你的梦里 
 照在河流中我喜欢的那一滴水里 
 &nbsp; 
 草露是一条河，草是一丝泥土 
 草在荣荣枯枯的生活中一次又一次失败着 
 草与草相接是一张网，能从故土铺向无尽头的远方 
 草从没想过，要在这个世间存留多久 
 &nbsp; 
 我在烦恼与痛苦中劳作，汗水与泪水照耀着前行中的我 
 我是一个草民，是一根无根的草，在易腐易生里，一秒又一秒地用一颗挣扎回乡的心，走完一春又一春 
 &nbsp; 
 我守着全世界的黑暗，从春天的泥土中醒来 
 我的梦想，只是挂在地球最边上的一座村庄 
 &nbsp; 
 每座坟墓，都埋着我的水 
 每座山岗，都有喂养我的碗 
 我是草，我是草上的梦露，我是大地最小最干净的孩子 
 &nbsp; 
 我的故乡太远，比所有的远多远 
 天空和大地，一个是我父亲，一个是我母亲 
 一块土，一块永恒的土，永远在我的根里 
 &nbsp; 
 小草啊小草，你为何扎根在大地的每一个角落 
 一朵花，一朵花开在了你的脸上 
 花仿佛是一朵火焰，燃烧着你走向远方 
 &nbsp; 
 我必将死，我必将在去往来生的路上，抱着一根草，靠着一把祖先传下来的锄头，慢慢在泥土中长出自己的骨头 
 &nbsp; 
 我是一个怀着一颗泥土心的人 
 我必将过完我平凡的一生 
 我泥土的心上在开着一朵花，在长着一根草 
 &nbsp; 
 一块泥土就是我的祖国 
 一块泥土就是我的灯 
 一个女人在一棵草下生了我 
 我在一棵草下必将埋葬这个女人 
 我在一棵草下必将过完我草的一生 
 &nbsp; 
 不要为一棵草悲伤 
 大地茫茫，我与草生长在天地中央 
 总有人受伤，总有人死亡 
 只要生活过，只要忙着、爱着、幸福着 
 只要一滴泪水，一缕月光，一根结米的小草 
 &nbsp; 
2010年10月28日写于岳麓山下
