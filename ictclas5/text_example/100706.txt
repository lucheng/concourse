视频文件格式众多，用户也是各有所爱。随着视频编码方式H.264渐成主流，视频文件格式领域也在悄然发生变化。据Sorenson Media发布的一份研究显示，MP4已成为跨越Web和移动两大平台的视频文件格式首选。 Sorenson Media是一家知名的视频编码解决方案公司，旗下的产品和服务被美国国内多家传媒集团所使用。因此，Sorenson Media的研究的调查对象主要面向视频领域的专业人士，一定程度上可以反映行视频业内对诸多文件格式的选择态度以及对它们应用前景的预期。调查结果显示，68%的专业人士经常在Web上使用MP4格式，移动端上的使用率也高达58%。而且，编码方式H.264的地位再一次得到证明，它是几乎五分之四的专业人士的编码压缩方式的首选。由Google大力支持的开源视频文件格式WebM并没有得到专业人士的过多青睐，只有5%的受访者在Web上使用它，移动端的使用者更是只有可怜的35人。 下面是受访者对Web上视频文件格式的选择态度： MP4->69% Flash->54% QuickTime->45% Windows Media->34% WebM->5% 注：大多数受访者选择的偏好视频文件格式不止一种，所以统计结果的百分比之和超过100。 下面是受访者对移动端上视频文件格式的选择态度： MP4->58% Flash->16% HTML5->16% Windows Media->9% WebM->3% Sorenson Media的研究还给出了专业人士对视频输入格式的选择态度。在这一方面，专业人士的选择面较为分散，并没有出现像MP4那样的“热门格式”。其中，苹果公司的QuickTime Reference Movie是最受欢迎的视频输入格式。 QuickTime Reference Movie->28% MOV->25% ProRes->10% AVI->9% DNxHD->8% MPEG-4->7% MPEG-2->3% FLV->2% WMV->2% 尽管现在移动视频风头正劲，但Web仍旧是最主要的视频发布渠道。下面是Sorenson Media给出的统计结果： Web->89% DVD->67% 移动端->46% 企业团体->45% 广播电视->43%
