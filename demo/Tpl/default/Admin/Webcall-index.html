365WebCall客服设置

免费版365在线客服开通步骤说明：
第一步、用户名：填好需要设置的在线客服登陆用户名可以是EMAIL
第二步、EMAIL：即365在线客服的主管理帐号，
第三步、密码：填上需要开设的客服管理密码
第四步、设好前边的三个填项后，点击右下角的"设置"，进行保存确认。
第五步、点击“开通客服帐号”成为365在线客服的免费用户。
